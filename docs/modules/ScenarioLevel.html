Local Functions

_changePlayerLevel (self, playerNumber, levelDifference)

Update the player's level with the given levelDifference and then calls _updateTotalPlayers to update the total player count.

  • If the player's level would go below zero then it will be set to one.
  • If the player's level would go above nine then it will be set to nine.

Parameters:

  • self self
  • playerNumber number is the index representing the color slots for Players
  • levelDifference number is the amount the player's level will change by.
_updateTotalPlayers (self)
Updates the total player count based on the TTS Players This is based on the number of players contained in players that are not null.
